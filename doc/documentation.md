# Documentation

## Installation

```bash
cargo install --path .
```

## Usage

### App Mode (TUI)

```bash
knowledge-index
```

Launches the full-screen interactive interface.

**Controls:**
- `q` / `Ctrl+C` / `Ctrl+D` — Exit the application
- Arrow keys — Navigate menus
- `Enter` — Select/confirm

### CLI Mode

```bash
knowledge-index [COMMAND] [OPTIONS]
```

Run `knowledge-index --help` for available commands.
